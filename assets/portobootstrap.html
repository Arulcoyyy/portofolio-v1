Portofolio bootstrap 5

portfolio Web pribadi yang menampilkan profil pengguna secara unik dan menarik. Dengan desain sederhana namun modern, situs ini berfungsi sebagai platform untuk memperkenalkan identitas, proyek, serta kemampuan yang dimiliki oleh pemiliknya.

Live Demo GitHub

Fitur Utama

  1. ✔ Profil Personal: Bagian utama menampilkan foto profil, nama, dan deskripsi singkat.
  2. ✔ Home: Halaman utama dengan informasi singkat pemilik website.
  3. ✔ About: Menjelaskan latar belakang atau keahlian pemilik.
  4. ✔ Projects: Showcase proyek atau karya.
  5. ✔ Desain Visual: Warna dominan ungu memberikan kesan elegan dan modern.
  6. ✔ Responsif di berbagai perangkat, baik desktop maupun mobile.
