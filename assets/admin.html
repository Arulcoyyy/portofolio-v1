Admin Diabtsweb

Admin diabtsweb merupakan dashboard administrasi yang dirancang untuk mengelola data pengguna, admin, dokter, dan riwayat pemeriksaan kesehatan terkait aplikasi diabetes. Tampilannya profesional dan minimalis, dengan navigasi yang rapi dan informatif. Yang memudahkan pemantauan dan mengelola data secara efisien, mendukung pengelolaan layanan kesehatan digital berbasis data.

Live Demo GitHub

Fitur Utama

  1. ✔ Web ini mendukung pengelolaan data pengguna, termasuk jumlah pengguna yang terdaftar dan riwayat cek gula darah mereka
  2. ✔ Mempermudah pencarian data gula darah berdasarkan kriteria tertentu untuk analisis yang akurat
  3. ✔ memungkinkan Anda menambahkan dokter baru ke dalam sistem dengan mudah
  4. ✔ Menampilkan jumlah admin yang terdaftar dan mempermudah pengelolaan akses admin ke sistem.
  5. ✔ Informasi akun ditampilkan di sudut kanan atas halaman, memberikan identitas yang jelas.
  6. ✔ Tampilan yang bersih dan estetik menggunakan elemen modern seperti card box untuk data penting.
